# 🤖 Robot Vacuum Cleaner Simulator

## 📌 Project Overview
This project demonstrates an Artificial Intelligence based Robot Vacuum Cleaner Simulator using the A* (A-star) search algorithm.

The robot intelligently navigates a grid environment, avoids obstacles, and cleans dirt efficiently.

---

## 🎯 Objectives
- Implement AI pathfinding
- Demonstrate A* Algorithm
- Simulate autonomous robot behavior
- Visualize intelligent decision-making

---

## 🧠 Algorithm Used
- A* (A-Star) Search Algorithm
- Manhattan Distance Heuristic
- Graph-based Search

---

## ⚙️ Technologies Used
- Python
- Pygame
- Artificial Intelligence Concepts

---

## 🖥️ System Requirements
- Python 3.x
- 4GB RAM minimum
- VS Code / PyCharm
